# Raytracer
Simple raytracer written in Java, using Swing to display rendered scenes.
# Features
- multithreading
- multisampling:
  - antialiasing
  - soft lighting
  - depth of field
- cameras:
  - orthogonal
  - perspective
- objects:
  - plane
  - sphere
- materials:
  - ambient
  - diffuse
  - specular
  - emissive
  - reflective
  - refractive
  - chess board
- lights:
  - point light
  - directional light
  - ambient light
